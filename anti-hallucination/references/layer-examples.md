# 各层执行示例Layer Examples

## Layer 1 示例 — 输入分析

**用户输入：**
> "2023年中国新能源汽车渗透率是多少？"

**Layer 1 执行：**
```
输入分析：
- 类型：事实查询
- 涉及：具体数字/年份/行业数据
- 风险等级：🔴 高风险
- 执行路径：全部9层
```

---

## Layer 3 示例 — 四类信息标注

**正确示范：**
```
新能源汽车渗透率在2023年有显著增长。（已知事实，广泛报道）
具体数字各机构统计口径不同，区间约在30%-35%之间。【待核实——建议查乘联会官方数据】
这一趋势预计将持续。【推断——基于政策方向和消费趋势】
```

**错误示范（禁止）：**
```
2023年中国新能源汽车渗透率为33.6%，同比增长12.3个百分点。
（❌ 给出精确数字但无来源，属于幻觉风险）
```

---

## Layer 4 示例 — 证据结构化

```
结论：新能源汽车渗透率在2023年超过30%
依据：乘联会、中汽协等机构均有相近报告
来源：乘联会官网月度销量报告（具体数值建议直接查询官方最新数据）
不确定性：统计口径不同（含/不含商用车），数值存在差异
```

---

## Layer 7 示例 — 幻觉扫描

**扫描过程（内部执行）：**
```
□ 数据风险：出现了"30%-35%"区间数据
  → 来源可靠吗？→ 系常识性范围，但建议用户核实精确值 ✓（加待核实标注）

□ 引用风险：提到"乘联会"
  → 真实存在的机构？→ 是，全国乘用车市场信息联席会 ✓

□ 案例风险：无具体公司案例 ✓
```

---

## Layer 9 示例 — 输出后审计

```
📋 自检报告：

以下内容可能未经证实：
1. "30%-35%区间" — 系综合多方报道的估算范围，非单一权威数据，建议查询乘联会官方公告
2. "趋势将持续" — 属推断，受政策变化影响

建议：如用于正式报告，请通过乘联会官网或中国汽车工业协会核实精确数字。
```
